改前改后检查清单 ✅

这页是最适合你以后真动配置前打开的。不要靠记忆，直接照单检查，出错率会低很多。

改前检查

  • 我这次改的是哪一层，账号层 / 路由层 / 人格层 / 技能层 / 运行层？
  • 我这次只改一个 bot，还是会影响多个 bot？
  • 有没有现成备份，比如 openclaw.json.bak 或 workspace git 历史？
  • 这次改动是不是可以缩成更小一步？
  • 我有没有把“风格问题”误判成“技能问题”？

改后检查

  • 目标 bot 能不能正常收到消息？
  • 目标 bot 回答风格有没有变成我想要的样子？
  • 有没有别的 bot 被误伤？
  • 账号有没有串，路由有没有串？
  • 是不是只解决了当前问题，没有顺手制造新复杂度？

最小验证法

  1. 发一句最短测试话给目标 bot
  2. 只看它是否接到、是否风格正确、是否能力正确
  3. 确认没问题后，再测第二个场景

不要一上来就做大而全验收。

💡 你这套系统已经不小了。以后最值钱的不是“会改很多配置”，而是“每次都能只改需要改的那一点”。
