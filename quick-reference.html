快速参考 ⚡

只保留最值得记住的那几条。

最核心的文件

  • ~/.openclaw/openclaw.json：总控台
  • bindings[]：路由总开关
  • bot-workspaces/<bot>/AGENTS.md：岗位规则
  • bot-workspaces/<bot>/SOUL.md：人格气质
  • bot-workspaces/<bot>/USER.md：用户偏好

最核心的判断

  • Bot 账号 ≠ Agent ≠ Workspace
  • 风格问题先看 workspace
  • 串号问题先看 binding
  • 能力没分开先看 skills allowlist
  • 越靠近 token / gateway，风险越高

最稳的改法

  1. 只改一层
  2. 只改一个 bot
  3. 改完马上做最小验证

最稳的排查法

  1. 先账号
  2. 后路由
  3. 再 workspace
  4. 最后技能和模型
⚠️ 如果你只记一条：不要把账号层、路由层、人格层、技能层混着一起改。
